暂无

## 功能

-   自动为项目生成`package.json`文件（如果没有的话）
-   询问式交互界面（基于 Inquirer.js）
-   自动复制模板文件（文件夹）到当前位置
-   可自行选择从模板中导入的文件、开发依赖和 npm 脚本
-   模板可扩展

## 快速开始

### 安装

```bash
# npm
npm install -g photinia

# yarn
yarn global add photinia

# pnpm
pnpm add -g photinia
```
